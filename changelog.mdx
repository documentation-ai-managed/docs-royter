## Upgrade Process

Follow these steps to upgrade Royter in your application or dashboard.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

  Review the changelog for breaking changes. Test in staging first.

  </Step>
  <Step title="Update SDK" icon="download">

  <CodeGroup tabs="npm,yarn">
  ````bash
  npm install @royter/sdk@latest
  ````
  ````bash
  yarn add @royter/sdk@latest
  ````
  </CodeGroup>

  </Step>
  <Step title="Migrate Code" icon="code">

  Update auth calls:

  ````javascript
  // Old
  const response = await fetch('https://api.example.com/v1/documents', {
    headers: { Authorization: `Bearer ${YOUR_OLD_TOKEN}` }
  });

  // New
  const token = await getNewJWT(); // Use /v1/auth/token
  const response = await fetch('https://api.example.com/v1/documents', {
    headers: { Authorization: `Bearer ${token}` }
  });
  ````

  </Step>
  <Step title="Deploy" icon="rocket">

  Deploy to production and monitor logs at `https://dashboard.example.com/logs`.

  </Step>
</Steps>
