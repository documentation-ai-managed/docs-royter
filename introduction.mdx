## Overview

Royter empowers teams to automate repetitive tasks and collaborate seamlessly. You gain access to an intuitive dashboard at `https://dashboard.example.com` where you manage workflows, integrations, and analytics in one place. Royter reduces manual effort by up to `50%`, letting you focus on high-value work.

Targeted at developers, product managers, and operations teams, Royter suits startups scaling operations or enterprises optimizing processes. You integrate it with tools like GitHub, Slack, and databases effortlessly.

<Callout kind="info">
Royter handles `>10,000` workflows daily across `500+` teams.
</Callout>

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com/signup` and create your account with email or Google.
  </Step>
  <Step title="Create Your First Workflow" icon="settings">
    Navigate to the Workflows tab and use the drag-and-drop builder.
  </Step>
  <Step title="Test with API" icon="play">
    Make a test call to verify setup.
    
    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const response = await fetch('https://api.example.com/v1/workflows/test', {
        method: 'POST',
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({ name: 'Test Workflow' })
      });
      const data = await response.json();
      console.log(data);
      ```
      ```bash
      curl -X POST https://api.example.com/v1/workflows/test \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{"name": "Test Workflow"}'
      ```
    </CodeGroup>
  </Step>
</Steps>
